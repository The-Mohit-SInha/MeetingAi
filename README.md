# AI Meeting-to-Action Management System

A modern, production-ready meeting management and action item tracking application built with React, TypeScript, Tailwind CSS, and Supabase.

## ✨ Features

- **Meeting Management**: Create, view, edit, and delete meetings with full details
- **Audio Recording & Transcription**:
  - Record from microphone with real-time transcription
  - Capture audio from browser tabs (Google Meet, Zoom, etc.)
  - Automatic transcription with Groq Whisper API (free, high-quality)
  - Fallback to local browser-based transcription
- **AI-Powered Analysis**: Automatic meeting summaries and action item extraction
- **Action Items**: Track tasks and action items from meetings
- **Participants**: Manage meeting participants and attendees
- **Analytics**: View statistics and insights on meetings and action items
- **Notifications**: Real-time notifications for important events
- **Calendar View**: Visualize meetings in a calendar format
- **Profile Management**: Update user profile and avatar
- **Settings**: Customize theme, compact mode, and notification preferences
- **Data Export**: Export meetings and action items to CSV or JSON

## 🛠️ Tech Stack

- **Frontend**: React 18 with TypeScript
- **Routing**: React Router v7
- **Styling**: Tailwind CSS v4 with custom design system
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **AI Transcription**: Groq Whisper API (free tier) + local Whisper fallback
- **AI Analysis**: Claude API (optional, for meeting summaries)
- **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and npm/pnpm
- Supabase account and project

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. Set up Supabase:
   - Create a Supabase project at https://supabase.com
   - Update `/utils/supabase/info.tsx` with your project credentials (auto-generated by Figma Make)
   - Run the database migration from `/supabase/migrations/20260402000000_initial_schema.sql` in the Supabase SQL Editor

4. **(Optional) Set up Groq for audio transcription**:
   - Get a free API key from [console.groq.com](https://console.groq.com/)
   - Add to your environment: `GROQ_API_KEY=your_key_here`
   - See [GROQ_SETUP.md](./GROQ_SETUP.md) for detailed instructions
   - Works without setup (falls back to local browser transcription)

5. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

6. Open http://localhost:5173 in your browser

## 🎙️ Audio Recording & Transcription

The app features powerful audio recording and transcription capabilities:

### Recording Modes

1. **Microphone Mode**:
   - Records your voice directly
   - Real-time transcription using Web Speech API
   - Works instantly, no setup required
   - Best for: voice memos, personal meetings

2. **Browser Tab Mode**:
   - Captures audio from any browser tab
   - Perfect for recording Google Meet, Zoom, or any web-based meeting
   - Automatic transcription after recording
   - No microphone needed - captures tab audio directly

### Transcription Services

The app uses a tiered approach for best quality:

1. **Groq Whisper API** (Primary, Free):
   - High-quality transcription
   - Fast processing (< 5 seconds)
   - Free tier: 14,400 requests/day
   - Setup: Add `GROQ_API_KEY` environment variable
   - See [GROQ_SETUP.md](./GROQ_SETUP.md)

2. **Local Browser Whisper** (Fallback, Free):
   - Runs entirely in your browser
   - No API keys needed
   - Works offline
   - Good quality, ~40MB model download on first use

### How It Works

1. Click "Record Audio" on the Meetings page
2. Choose your source (Microphone or Browser Tab)
3. For tab mode: Select the tab and check "Share tab audio"
4. Record your meeting
5. Stop recording - transcription happens automatically
6. Meeting is saved with full transcript and auto-generated summary
